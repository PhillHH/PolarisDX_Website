# PolarisDX — Projektknowledge

Wissensbasis für die strategische Überarbeitung der PolarisDX-Website.

## Inhalt

- `config/` — Build- und TypeScript-Konfiguration
- `server/` — SSR-Server und Mail-Service
- `app-shell/` — Routing und App-Entry
- `i18n/` — Internationalisierung-Setup
- `components/` — UI-Komponenten (layout, sections, ui, seo)
- `pages/` — Seiten mit redaktionellem Content
- `locales/` — DE- und EN-Übersetzungen
- `deploy/` — Deployment-Konfiguration
- `AUDIT-REPORT.md` — Kurzfassung des Website-Audits
